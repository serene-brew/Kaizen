<table align=center>

<tr><th> In app </th><th> In player </th></tr>
<tr><td>
  
| Operation       | Keybind       |
|-----------------|----------------|
| Toggle b/w tabs |     `tab`      |
| Focus on Input  | `shift` + `1`  |
| Focus on Results| `shift` + `2`  |
| Focus on SUB ep | `shift` + `3`  |
| Focus on DUB ep | `shift` + `4`  |
| Exit the app    |     `esc`      |

| Operation       | Keybind       |
